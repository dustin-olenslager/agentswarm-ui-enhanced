## Upstream Sync Policy

This fork maintains synchronization with [andrewcai8/agentswarm](https://github.com/andrewcai8/agentswarm) while adding significant UI and developer experience enhancements. 

### Sync Strategy
- **Automated Monitoring** - CI checks for upstream changes daily
- **Manual Review** - All upstream changes reviewed before integration  
- **UI Preservation** - Our UI enhancements are preserved during syncs
- **Contribution Back** - UI improvements contributed back to upstream when appropriate

- **UI-First Design** - Dashboard and visualization as primary features
